长亭雷池WAF使用教程

· 2023-04-22 · # 长亭 # WAF

记录一下长亭雷池WAF的使用教程

项目地址: https://github.com/chaitin/safeline

安装

安装前需要确保Docker和Compose V2已经安装且以达到版本要求

docker info # >= 20.10.6
docker compose version # >= 2.0.0

如果未安装Compose V2, 可以使用该命令安装sudo apt update && sudo apt install docker-compose-plugin
依赖安装完成后执行

# 新建文件夹
mkdir -p safeline && cd safeline
# 安装
curl -kfLsS https://waf-ce.chaitin.cn/release/latest/setup.sh | bash
# 启动
sudo docker compose up -d

安装完成后可以访问 https://:9443 进行配置, 使用TOTP APP扫码然后登录进去

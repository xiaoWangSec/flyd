Arco Pro上手历程

· 2023-05-21 · # 开发 # Vue

记录一下学习Arco Pro的过程

安装

npm -i -g arco-cli

利用脚手架新建项目

arco init hello-arco-pro

注意: 如果初始化过程中遇到了奇怪的报错, 可以做如下尝试

  • 安装pnpm并换国内源, 或者还需要pnpm init
  • 使用代理工具例如Clash并开TUN模式

跑起来

cd hello-arco-pro
npm run dev

构建命令

npm run build

更多run命令可以参考目录下package.json文件

目录结构

├── README.md
├── package.json
├── index.html
├── src
│   ├── api  # 请求接口
│   ├── assets  # 静态资源
│          └── style 全局样式
│   ├── components  # 通用业务组件
│   ├── config  # 全局配置(包含echarts主题)
│          └── settings.json  # 配置文件
│   ├── directives # 指令集（如需，可自行补充）
│   ├── filters # 过滤器（如需，可自行补充）
│   ├── hooks # 全局hooks
│   ├── layout  # 布局
│   ├── locale  # 国际化语言包
│   ├── mock  # 模拟数据
│   ├── views  # 页面模板
│   ├── router # 路由配置
│   ├── store  # 状态管理中心
│   ├── types  # Typescript 类型
│   └── utils  # 工具库
│   └── App.vue  # 视图入口
│   └── main.ts  # 入口文件
└── tsconfig.json

路由

基本配置

// 在本例子中，页面最终路径为 /dashboard/workplace
export default {
 path: 'dashboard',
 name: 'dashboard', // 路由名称
 component: () => import('@/views/dashboard/index.vue'),
 meta: {
   locale: 'menu.dashboard',
   requiresAuth: true,
   icon: 'icon-dashboard',
 },
 children: [
   {
     path: 'workplace',
     name: 'workplace',
     component: () => import('@/views/dashboard/workplace/index.vue'),
     meta: {
       locale: 'menu.dashboard.workplace',
       requiresAuth: true,
       roles: ['admin'],
       hideInMenu: false,
     },
   },
 ],
};

新建一个一级菜单

类似二级菜单

  • 在/src/routes/modules下新建一个users.ts的文件
import { DEFAULT_LAYOUT } from '../base';
import { AppRouteRecordRaw } from '../types';

const USERS: AppRouteRecordRaw = {
  path: '/users',
  name: 'users',
  component: DEFAULT_LAYOUT,
  meta: {
    locale: 'menu.users',
    requiresAuth: true,
    icon: 'icon-h1',
    order: 0,
  },
  children: [
    {
      path: 'users-list',
      name: 'UserList',
      component: () => import('@/views/user/users-list/index.vue'),
      meta: {
        locale: 'menu.users.users-list',
        requiresAuth: true,
        roles: ['admin'],
      },
    },
  ],
};

export default USERS;
  • 然后去/views/user/users-list/新建index.vue
<template>
  <span>User list here!</span>
</template>

<script>
  export default {
    name: 'index.vue',
  };
</script>

<style scoped></style>

  ......
  'menu.users': '用户管理',
  'menu.users.users-list': '用户列表',
